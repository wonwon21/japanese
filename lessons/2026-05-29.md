# Day 2 세션 (2026-05-29, Session 2)

## 신규 단어 (7개)

### 숫자 (Native Japanese, 1-5)
1. 一(いち) - 1 | 예: 一つください (하나 주세요)
2. 二(に) - 2 | 예: 二人です (둘입니다)
3. 三(さん) - 3 | 예: 三つ (셋) | 메모: ㅁ받침 → ん
4. 四(し/よん) - 4 | 예: 四月です (4월) | 메모: よん 발음 많이 씀
5. 五(ご) - 5 | 예: 五時です (5시)

### い形容詞
6. 大きい(おおきい) - 크다 | 예: 大きいです (큽니다)
7. 小さい(ちいさい) - 작다 | 예: 小さいです (작습니다)

## 신규 문법
- **い形容詞 + です**: 순수 형용사가 이미 -い로 끝나므로 そのまま です 붙임
  - 例: 大きいです = 큽니다
  - 부정: 大きくないです = 크지 않습니다
  - 명사 수식: 大きい本 = 큰 책
  - 한국어 "크다"와 달리, 일본어는 순수 형용사 구조

## 작문 결과
1. これは本です。✓ (이것은 책입니다)
2. 大きいです。✓ (큽니다)
3. 本は小さいです。✓ (책은 작습니다)

## 오답 및 학습 메모
- 오답 없음
- 주요 질문: 5つあります (5개 있다) 구문 설명 → Native숫자 체계 + あります 문법
- Native vs Sino-Japanese 숫자 차이 학습: 
  - Native (ひとつ~ここのつ) = 일상 세기, 1-10 한정
  - Sino-Japanese (一~) = 공식 표현, Counter 필수, 무한대
  - 예: 本が5つ vs 本3冊(より自然)

## 헷갈린 가나 (Day 2)
- ね vs れ: 정확히 구분함 ✓
- わ: 복습

# Day 3 세션 (2026-05-29, Session 3)

## 신규 단어 (7개)

### 동사
1. 食べる(たべる) — 타벨루 / 먹다 | 예: 私は毎日ご飯を食べます
2. 飲む(のむ) — 노무 / 마시다 | 예: 水を飲みます

### 장소/물건
3. 家(いえ) — 이에 / 집 | 예: 家に行きます
4. パン — 판 / 빵 | 예: パンを食べます
5. 水(みず) — 미즈 / 물 | 예: 冷たい水です
6. 公園(こうえん) — 코우엔 / 공원 | 예: 公園に行きます
7. 図書館(としょかん) — 토쇼칸 / 도서관 | 예: 図書館は静かです

## 신규 문법
- **動詞のます形**: [동사 어간] + ます
  - 의미: 존댓말 현재 (한국어 합니다체)
  - -う그룹 음편 규칙: う단 → い단 (모든 -u동사 적용)
    - く→き, ぐ→ぎ, つ→ち, む→み, ぬ→に, ぶ→び, す→し, る→り
  - 예: 飲む → 飲みます, 行く → 行きます

## 작문 결과
わたしはきょうご飯を食べます、コーヒーものみます、学校に行きます。
- ✓ 동사 ます형 2개 (食べます/飲みます) — 음편 정확
- ✓ を조사 (목적어) — 정확
- ✓ に조사 (장소) — 정확
- ✓ 새 어휘 자연스러운 확장 (コーヒー)

## 오답 및 학습 메모
- 오답 없음
- 주요 질문: "노무 → 노미마스"와 "이쿠 → 이키마스" 음편 차이 원인 → う단→い단 통일 규칙 확인
- 발음 정정: ご飯(ごはん) = "고한" (한국어 자연음운)
- 피드백 요청: 신규 단어 시 한글 발음 병기 (학습 효율성) → 반영됨
