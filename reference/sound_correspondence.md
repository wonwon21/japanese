# 한국어 ↔ 일본어 한자음 대응 패턴

학습하면서 발견하는 음운 대응 규칙을 누적 기록한다.
새 한자어 단어를 만날 때 이 표를 참조해 발음을 추론·검증한다.

## 받침(종성) 대응

| 한국어 받침 | 일본어 변환 | 예시 |
|-------------|-------------|------|
| ㄱ | っ (촉음) + k행 | 학교 がっこう / 약속 やくそく / 국가 こっか |
| ㄴ | ん | 간단 かんたん / 안심 あんしん |
| ㅁ | ん | 신문 しんぶん / 음악 おんがく |
| ㄹ | つ / ち | 일 いち / 발 はつ / 출 しゅつ |
| ㅂ | う / つ | 잡 ざつ / 입 にゅう / 합 ごう |
| ㅇ | う / い (장음화) | 공 こう / 성 せい / 정 てい |

## 모음(중성) 대응

| 한국어 | 일본어 | 예시 |
|--------|--------|------|
| ㅏ | あ | 가 か |
| ㅗ | お | 보 ほ |
| ㅜ | う | 무 む |
| ㅣ | い | 시 し |
| ㅓ | お/え | 서 しょ |

## 초성(자음) 대응 메모

- ㄱ → か행 / が행
- ㅅ → さ행
- ㅈ → さ행/ざ행 (자 → ざ, 주 → しゅ)
- ㅎ → か행/が행 (한자음에서 ㅎ이 か로 가는 경우 많음: 화 か, 회 かい)
