# JLPT N5 문법 순서 (학습 진행 순서)

매일 1개씩 진행. Claude는 이 리스트의 미학습 항목 중 가장 위에 있는 것을 다음 문법으로 출제한다.

## Phase 1: 기본 문장 구조 (Day 1~10)
- [ ] です (긍정 단정문)
- [ ] ではありません / じゃありません (부정문)
- [ ] ですか (의문문)
- [ ] は (주제 조사)
- [ ] が (주어 조사)
- [ ] も (~도)
- [ ] の (소유·수식)
- [ ] これ・それ・あれ (지시대명사)
- [ ] この・その・あの + 명사
- [ ] ここ・そこ・あそこ (장소)

## Phase 2: 동사 기초 (Day 11~25)
- [ ] 동사 ます형 (긍정)
- [ ] 동사 ません (부정)
- [ ] 동사 ました (과거)
- [ ] 동사 ませんでした (과거 부정)
- [ ] を (목적격 조사)
- [ ] に (시간·도착점)
- [ ] へ (방향)
- [ ] で (장소·수단)
- [ ] と (~와/과, 동반)
- [ ] から / まで (~부터 / ~까지)
- [ ] ましょう / ませんか (권유)
- [ ] 동사 분류 (1그룹/2그룹/3그룹)
- [ ] て형 활용 규칙
- [ ] ~ています (진행·상태)
- [ ] ~てください (요청)

## Phase 3: 형용사 (Day 26~35)
- [ ] い형용사 긍정 (현재)
- [ ] い형용사 부정 (~くないです)
- [ ] い형용사 과거 (~かったです)
- [ ] い형용사 과거 부정 (~くなかったです)
- [ ] な형용사 긍정 (~です)
- [ ] な형용사 부정 (~じゃありません)
- [ ] 형용사 + 명사 수식
- [ ] 형용사 て형 연결
- [ ] とても / あまり (정도 부사)
- [ ] 비교: より / ほうが

## Phase 4: 확장 표현 (Day 36~50)
- [ ] ~たい (희망)
- [ ] ~たことがあります (경험)
- [ ] ~ことができます (가능)
- [ ] 동사 가능형
- [ ] ~ながら (동시 동작)
- [ ] ~前に / ~後で (전·후)
- [ ] ~とき (~할 때)
- [ ] ~から (이유)
- [ ] ~ので (이유, 부드러움)
- [ ] ~でしょう (추측)
- [ ] ~なければなりません (의무)
- [ ] ~てもいいです (허가)
- [ ] ~てはいけません (금지)
- [ ] 수량 표현 (조수사: 個・本・枚・人・つ)
- [ ] 시간 표현 (時・分・~時間)

(완료 시 N5 문법 약 50개. 이후 N4 리스트로 전환)
