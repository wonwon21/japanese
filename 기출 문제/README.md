출처: 일본국제교류기금 JLPT 공식 사이트 (jlpt.jp) — 학습자용 무료 공개 자료 전량.

## 폴더 구성

```
기출 문제/
├── N5/  ── 문자어휘·문법·독해·정답·답안지 PDF
│   └── 청해/ ── 청해 문제지 + 스크립트 + 음원(mp3)
├── N4/  (동일 구성)
└── N3/  (동일 구성)
```

## 수록 자료 (레벨당 3세트)

| 세트 | 연도 | 내용 |
|------|------|------|
| 공식문제집 1집 | 2012 | 실제 출제 문제에서 추린 공식 문제집. 과목별 PDF + 청해 음원 4~5개 |
| 공식문제집 2집 | 2018 | 2012~2017년 실제 출제분 기반 2탄. 구성 동일 |
| 신시험 예제 | 2009 | 현행 시험 도입 때 공개된 견본 문제 + 청해 음원 |

파일명 규칙: `레벨_자료명_연도_과목.pdf` (예: `N5_공식문제집_1집_2012_문자어휘.pdf`)

## ⚠️ "몇년 몇차" 실제 기출이 없는 이유

JLPT는 **2010년 개편 이후 회차별 실제 시험지를 공개하지 않음** (문제 재사용 정책).
따라서 "2023년 1차" 같은 회차별 원본은 합법적으로 존재하지 않고,
인터넷에 도는 회차별 스캔본은 전부 무단 유출본(저작권 침해)이라 수집하지 않았다.

위 공식문제집 1·2집이 **실제 출제 문제를 공식적으로 묶어낸 유일한 기출 자료**다.

## 청해 학습법

1. 음원(mp3) 듣고 문제지 PDF로 풀기
2. 정답 확인
3. 스크립트 PDF 보면서 다시 듣기 (받아쓰기 추천)
